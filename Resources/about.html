1. Меню текстового редактора

Пункт "Файл"

Позволяет создавать, открывать и сохранять документы, а также закрывать вкладки и выходить из приложения.

Пункт "Правка"

Содержит стандартные операции редактирования: копировать, вырезать, вставить, удалить, отменить, повторить.

Пункт "Текст"

Содержит теоритическую информацию о работе синтаксического анализатора.

Пункт "Пуск"

Запускает анализ кода.

Пункт "Справка"

Открывает данное руководство пользователя.

Пункт "Настройки"

Позволяет переключить язык интерфейса программы, настроить шрифты для окон ввода и вывода.

Пункт "Справка"

Позволяет вызвать справочную информацию для пользователя, а также сведения о программе.

2. Панель инструментов

На панели расположены кнопки быстрого доступа:

  • Создать новый документ
  • Открыть документ
  • Сохранить текущий документ
  • Отменить последнее действие
  • Повторить последнее действие
    • ъ
  • Копировать выделенный фрагмент
  • Вырезать выделенный фрагмент
  • Вставить выделенный фрагмент
  • Запуск анализатора
  • Вызов справки
  • Отображение информации о программе

3. Дополнительные возможности

  • Базовая подсветка синтаксиса
  • Поддержка работы с несколькими вкладками
  • Изменение размеров текста в окне редактирования и окне вывода результатов
  • Нумерация строк в окне редактирования текста
  • Наличие строки состояния для отображения текущей информации о состоянии работы приложения
  • Горячие клавиши для быстрых команд
  • Drag-and-Drop перетаскивание файлов
  • Интернационализация (локализация интерфейса)
